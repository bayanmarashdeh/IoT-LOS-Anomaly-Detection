# IoT Line-of-Sight (LoS) and Machine Anomaly Detection

## Project Overview

This project was developed as part of an Internet of Things (IoT) course.

The project consists of three main phases:

### Step 1
- Deploy Access Points (APs), IoT sensors, and blockers.
- Create a 3D visualization of the network.
- Detect whether the Line-of-Sight (LoS) between each Access Point and IoT device is blocked.

### Step 2
- Process the MIMII dataset.
- Extract audio features from machine sounds.
- Train a Machine Learning model to classify normal and abnormal machine conditions.

### Step 3
- Evaluate the communication performance and anomaly detection results.
- Display the obtained results and performance metrics.

## Technologies Used

- Python
- NumPy
- Matplotlib
- Librosa
- Scikit-learn

## Dataset

This project uses the MIMII dataset for industrial machine anomaly detection.
